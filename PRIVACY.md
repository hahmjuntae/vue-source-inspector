# Privacy Policy

Last updated: April 9, 2026

Vue Source Inspector does not collect, transmit, sell, or share personal data with any remote server.

What the extension does:

- Inspects Vue-rendered elements on pages you open.
- Reads component metadata already exposed by the target page's Vue runtime.
- Stores local extension preferences such as the selected editor and inferred project root using Chrome local storage.

What the extension does not do:

- It does not send browsing data to any backend.
- It does not sync data to a cloud service.
- It does not use analytics, ads, or tracking scripts.
- It does not download and execute remote code.

Stored local data:

- `vsiEditorKind`
- `vsiInferredProjectRoot`

This local data is used only to improve the DevTools workflow on your machine.

If the extension behavior changes in the future, this policy should be updated before publication.
